ENQUIRY TRACKER
Setup Required
This dashboard reads live email data from your Microsoft 365 inbox. You'll need to create a free Azure App Registration (5 minutes), then add the details to the CONFIG block in this file.
1
Go to portal.azure.com → Azure Active Directory → App Registrations → New Registration
2
Name it "Techlam Enquiry Tracker". Set Redirect URI to Single-page application (SPA) and paste the URL where this file is hosted (e.g. the URL where this file is hosted)
3
Under API Permissions, add delegated permissions: Mail.Read, User.Read, Sites.ReadWrite.All -- then click Grant admin consent
4
Copy the Application (client) ID and Directory (tenant) ID into the CONFIG block at the top of this file's <script> section
5
Set MAILBOX_EMAIL to your shared estimating inbox address (e.g. tenders@techlam.co.nz) or 'me' for personal inbox
6
Set SHAREPOINT_SITE to your SharePoint hostname and site path (e.g. techlam.sharepoint.com:/sites/Estimating). The notes list TLEnquiryNotes will be auto-created on first save -- all users will see the same notes in real time.
ⓘ Host this file at any https:// URL. The Redirect URI in Azure must exactly match the URL you open the file from.
